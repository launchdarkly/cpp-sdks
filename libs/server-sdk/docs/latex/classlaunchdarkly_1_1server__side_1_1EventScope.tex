\doxysubsection{Detailed Description}
\doxylink{classlaunchdarkly_1_1server__side_1_1EventScope}{Event\+Scope} is responsible for forwarding events to an IEvent\+Processor. If the given interface is nullptr, then events will not be forwarded at all.
