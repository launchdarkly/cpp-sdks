\doxysubsection{Detailed Description}
Represents a Launch\+Darkly-\/flavored Semantic Version v2.

Semantic versions can be compared using ==, $<$, and $>$.

The main difference from the official spec is that missing minor and patch versions are allowed, i.\+e. \char`\"{}1\char`\"{} means \char`\"{}1.\+0.\+0\char`\"{} and \char`\"{}1.\+2\char`\"{} means \char`\"{}1.\+2.\+0\char`\"{}.
