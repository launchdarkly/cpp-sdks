\doxysubsection{Detailed Description}
\mbox{\hyperlink{classlaunchdarkly_1_1server__side_1_1data__systems_1_1LazyLoad}{Lazy\+Load}} implements a Data System that pulls data from a persistent store on-\/demand. It is intended for use cases where holding an entire environment in memory isn\textquotesingle{}t desired, or for uses that disallow direct communication with Launch\+Darkly servers.

\mbox{\hyperlink{classlaunchdarkly_1_1server__side_1_1data__systems_1_1LazyLoad}{Lazy\+Load}} is able to remain efficient because it caches responses from the store. Over time, data becomes stale causing the system to refresh data.
