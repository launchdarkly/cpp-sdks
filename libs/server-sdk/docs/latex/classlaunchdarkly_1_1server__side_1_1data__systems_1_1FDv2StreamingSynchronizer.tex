\doxysubsection{Detailed Description}
FDv2 streaming synchronizer. Opens an SSE connection to the FDv2 streaming endpoint and translates the push-\/based event stream into the pull-\/based \mbox{\hyperlink{classlaunchdarkly_1_1server__side_1_1data__interfaces_1_1IFDv2Synchronizer_a24a4d0d4a77f467eaf55a5016a1bc9bc}{IFDv2\+Synchronizer\+::\+Next()}} interface.

Threading model\+: \mbox{\hyperlink{classlaunchdarkly_1_1server__side_1_1data__systems_1_1FDv2StreamingSynchronizer_a4d43ebac197ab33f07e6553466262fe7}{Next()}} should only be called once at a time. \mbox{\hyperlink{classlaunchdarkly_1_1server__side_1_1data__systems_1_1FDv2StreamingSynchronizer_a30c355eb3f40bebb6a6a3b46786df157}{Close()}} may be called concurrently with \mbox{\hyperlink{classlaunchdarkly_1_1server__side_1_1data__systems_1_1FDv2StreamingSynchronizer_a4d43ebac197ab33f07e6553466262fe7}{Next()}}. This object may be safely destroyed once no call to \mbox{\hyperlink{classlaunchdarkly_1_1server__side_1_1data__systems_1_1FDv2StreamingSynchronizer_a4d43ebac197ab33f07e6553466262fe7}{Next()}} or \mbox{\hyperlink{classlaunchdarkly_1_1server__side_1_1data__systems_1_1FDv2StreamingSynchronizer_a30c355eb3f40bebb6a6a3b46786df157}{Close()}} is in progress.

Returns a Future that resolves with the next result once it is available.

On the first call, the synchronizer starts its underlying connection. Subsequent calls continue reading from the same connection.
