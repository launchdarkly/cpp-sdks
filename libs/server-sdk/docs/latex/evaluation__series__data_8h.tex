\doxysubsection{Detailed Description}
C bindings for hook data passed between evaluation stages. 

Evaluation\+Series\+Data is mutable data that hooks can use to pass information from before\+Evaluation to after\+Evaluation. This is useful for\+:
\begin{DoxyItemize}
\item Storing timing information
\item Passing span contexts for distributed tracing
\item Accumulating custom metrics
\end{DoxyItemize}

LIFETIME AND OWNERSHIP\+:
\begin{DoxyItemize}
\item Data returned from hook callbacks transfers ownership to the SDK
\item Data received in callbacks can be modified and returned (ownership transfer)
\item Keys are copied by the SDK
\item Values retrieved with Get\+Value() are temporary -\/ valid only during callback
\item Do not call \mbox{\hyperlink{value_8h_a977c046a68f9451f14d5b9a35bbb0292}{LDValue\+\_\+\+Free()}} on values retrieved with Get\+Value()
\item Values stored with Set\+Value() are copied into the data object
\item Pointers (void$\ast$) lifetime is managed by the application
\end{DoxyItemize}

BUILDER PATTERN\+: To modify data, use \mbox{\hyperlink{evaluation__series__data_8h_a210b4a9ae1ff924ba3ecc05f876416b4}{LDEvaluation\+Series\+Data\+\_\+\+New\+Builder()}} to create a builder, make changes, then build a new data object.
