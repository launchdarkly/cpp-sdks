\doxysubsection{Detailed Description}
Internal layer between the evaluator and a customer-\/provided \mbox{\hyperlink{classlaunchdarkly_1_1server__side_1_1integrations_1_1IBigSegmentStore}{integrations\+::\+IBig\+Segment\+Store}}. 

Adds context-\/key hashing, an LRU membership cache, a background task that polls the store\textquotesingle{}s metadata to track availability and staleness, and a status-\/change broadcaster.

Construct with std\+::make\+\_\+shared and call \mbox{\hyperlink{classlaunchdarkly_1_1server__side_1_1data__components_1_1BigSegmentStoreWrapper_a9f31e96e9d02193705a32877af37fc01}{Start}} once to begin background polling; polling stops when the wrapper is destroyed.

Thread-\/safe\+: \mbox{\hyperlink{classlaunchdarkly_1_1server__side_1_1data__components_1_1BigSegmentStoreWrapper_a1596bdc4e1d2d22ad6afcc25bca21969}{Get\+Membership}}, \mbox{\hyperlink{classlaunchdarkly_1_1server__side_1_1data__components_1_1BigSegmentStoreWrapper_a4e3f6262ba1e94a5d1f00cb3a0bc4e7f}{Get\+Status}}, and \mbox{\hyperlink{classlaunchdarkly_1_1server__side_1_1data__components_1_1BigSegmentStoreWrapper_a454e08df69d4dc9ed21ac66fce133e9b}{On\+Status\+Change}} may be called concurrently from any number of evaluation threads, including while the background poll runs on the executor. The store is only ever queried with the relevant lock released, so a slow store never blocks unrelated callers.
