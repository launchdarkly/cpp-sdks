\doxysubsection{Detailed Description}
FDv2 polling synchronizer. Repeatedly polls the FDv2 polling endpoint at a configurable interval.

The caller passes the current selector into each \mbox{\hyperlink{classlaunchdarkly_1_1server__side_1_1data__systems_1_1FDv2PollingSynchronizer_a6d2cd5d8f7411d73f8845dee16bf95a8}{Next()}} call, allowing the orchestrator to reflect applied changesets without any shared state.

Threading model\+: \mbox{\hyperlink{classlaunchdarkly_1_1server__side_1_1data__systems_1_1FDv2PollingSynchronizer_a6d2cd5d8f7411d73f8845dee16bf95a8}{Next()}} should only be called once at a time. \mbox{\hyperlink{classlaunchdarkly_1_1server__side_1_1data__systems_1_1FDv2PollingSynchronizer_aae6fa611e02ab111a617c7d3e9fcee6b}{Close()}} may be called concurrently with \mbox{\hyperlink{classlaunchdarkly_1_1server__side_1_1data__systems_1_1FDv2PollingSynchronizer_a6d2cd5d8f7411d73f8845dee16bf95a8}{Next()}}. This object may be safely destroyed once no call to \mbox{\hyperlink{classlaunchdarkly_1_1server__side_1_1data__systems_1_1FDv2PollingSynchronizer_a6d2cd5d8f7411d73f8845dee16bf95a8}{Next()}} or \mbox{\hyperlink{classlaunchdarkly_1_1server__side_1_1data__systems_1_1FDv2PollingSynchronizer_aae6fa611e02ab111a617c7d3e9fcee6b}{Close()}} is in progress.
