\doxysubsection{Detailed Description}
Hook structure containing callback function pointers. 

USAGE\+:
\begin{DoxyEnumerate}
\item Allocate an \mbox{\hyperlink{structLDServerSDKHook}{LDServer\+SDKHook}} struct (stack or heap)
\item Call \mbox{\hyperlink{hook_8h_a2d30cbc6e7ee6d129231b6f36cc608b9}{LDServer\+SDKHook\+\_\+\+Init()}} to initialize it
\item Set the Name field (required, UTF-\/8 encoded, null-\/terminated)
\item Set any callback function pointers you need (NULL if not used)
\item Set User\+Data to application-\/specific context (NULL if not needed)
\item Register with \mbox{\hyperlink{server-sdk_2include_2launchdarkly_2server__side_2bindings_2c_2config_2builder_8h_a5f1fa2661cd1833efe90d29e338502a6}{LDServer\+Config\+Builder\+\_\+\+Hooks()}}
