\doxysubsection{Detailed Description}
FDv2 polling initializer. Makes a single HTTP GET to the FDv2 polling endpoint, parses the response via the FDv2 protocol state machine, and returns the result.

Threading model\+: \mbox{\hyperlink{classlaunchdarkly_1_1server__side_1_1data__systems_1_1FDv2PollingInitializer_a83b3bf96e8c7f6018aef8727b4dc0fe1}{Run()}} should only be called once at a time. \mbox{\hyperlink{classlaunchdarkly_1_1server__side_1_1data__systems_1_1FDv2PollingInitializer_a926234985ff2665d09d8eaaa873dcda1}{Close()}} may be called concurrently with \mbox{\hyperlink{classlaunchdarkly_1_1server__side_1_1data__systems_1_1FDv2PollingInitializer_a83b3bf96e8c7f6018aef8727b4dc0fe1}{Run()}}. This object may be safely destroyed once no call to \mbox{\hyperlink{classlaunchdarkly_1_1server__side_1_1data__systems_1_1FDv2PollingInitializer_a83b3bf96e8c7f6018aef8727b4dc0fe1}{Run()}} or \mbox{\hyperlink{classlaunchdarkly_1_1server__side_1_1data__systems_1_1FDv2PollingInitializer_a926234985ff2665d09d8eaaa873dcda1}{Close()}} is in progress.
