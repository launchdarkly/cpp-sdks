\doxysubsection{Detailed Description}
This class is meant only to record direct prerequisites of a flag. That is, although it will be passed events for all prerequisites seen during an evaluation via Send\+Async, it will only store those that are a direct prerequisite of the parent flag passed in the constructor.

As a future improvement, it would be possible to unify the \doxylink{classlaunchdarkly_1_1server__side_1_1EventScope}{Event\+Scope} mechanism currently used by the Evaluator to send events with a class similar to this one, or to refactor the Evaluator to include prerequisite information in the returned \doxylink{classlaunchdarkly_1_1EvaluationDetail}{Evaluation\+Detail} (or a new Result class, which would be a composite of the \doxylink{classlaunchdarkly_1_1EvaluationDetail}{Evaluation\+Detail} and a vector of prerequisites.)
