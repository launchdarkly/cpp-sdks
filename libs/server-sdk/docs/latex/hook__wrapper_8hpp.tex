\doxysubsection{Detailed Description}
C++ wrapper that bridges C hook callbacks to C++ Hook interface. 

This internal class captures C callback function pointers and user data, then forwards hook method calls from the C++ Hook interface to the C callbacks.

OWNERSHIP MODEL\+:
\begin{DoxyItemize}
\item The wrapper is created as a shared\+\_\+ptr and registered with Config\+Builder
\item The wrapper copies the Name string during construction
\item User\+Data lifetime is managed by the C application
\item Function pointers are copied and assumed valid for the SDK lifetime 
\end{DoxyItemize}
