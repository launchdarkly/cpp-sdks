\doxysubsection{Detailed Description}
Defines a continuous data source that produces a stream of results. Used during the synchronization phase of FDv2, after initialization is complete.

The stream is started lazily on the first call to \mbox{\hyperlink{classlaunchdarkly_1_1server__side_1_1data__interfaces_1_1IFDv2Synchronizer_a24a4d0d4a77f467eaf55a5016a1bc9bc}{Next()}}. The synchronizer runs until \mbox{\hyperlink{classlaunchdarkly_1_1server__side_1_1data__interfaces_1_1IFDv2Synchronizer_aa50569af87e47e3f0eb7308cbbe73514}{Close()}} is called.

Returns a Future that resolves with the next result once it is available.

On the first call, the synchronizer starts its underlying connection. Subsequent calls continue reading from the same connection.
