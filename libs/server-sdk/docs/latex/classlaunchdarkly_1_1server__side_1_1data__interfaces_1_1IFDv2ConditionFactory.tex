\doxysubsection{Detailed Description}
Builds new \mbox{\hyperlink{classlaunchdarkly_1_1server__side_1_1data__interfaces_1_1IFDv2Condition}{IFDv2\+Condition}} instances on demand. Each call to Build() produces a fresh condition with no prior state.

Implementations must be thread-\/safe\+: Build and Get\+Type may be called from any thread.
