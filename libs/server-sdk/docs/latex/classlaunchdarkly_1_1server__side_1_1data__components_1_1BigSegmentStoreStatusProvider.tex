\doxysubsection{Detailed Description}
Adapts a \mbox{\hyperlink{classlaunchdarkly_1_1server__side_1_1data__components_1_1BigSegmentStoreWrapper}{Big\+Segment\+Store\+Wrapper}} to the public \mbox{\hyperlink{classlaunchdarkly_1_1server__side_1_1IBigSegmentStoreStatusProvider}{IBig\+Segment\+Store\+Status\+Provider}}, converting the internal status struct to the public type. 

Holds a null wrapper when Big Segments are not configured; in that case it reports the store as unavailable and not stale, and registered listeners never fire.

Thread-\/safe\+: delegates to the wrapper, which is itself thread-\/safe.
