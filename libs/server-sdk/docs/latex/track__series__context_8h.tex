\doxysubsection{Detailed Description}
C bindings for read-\/only track context passed to after\+Track hooks. 

Track\+Series\+Context provides information about a track event to hook callbacks. Most data is read-\/only and valid only during the callback execution.

LIFETIME AND OWNERSHIP\+:
\begin{DoxyItemize}
\item All context parameters are temporary -\/ valid only during callback
\item Do not store pointers to context, key, or environment ID strings
\item Data retrieved with \mbox{\hyperlink{track__series__context_8h_a3ced697a09eb36b4950ec5147ecbc3b7}{LDTrack\+Series\+Context\+\_\+\+Data()}} is temporary -\/ valid only during callback
\item Do not call \mbox{\hyperlink{value_8h_a977c046a68f9451f14d5b9a35bbb0292}{LDValue\+\_\+\+Free()}} on data retrieved with \mbox{\hyperlink{track__series__context_8h_a3ced697a09eb36b4950ec5147ecbc3b7}{LDTrack\+Series\+Context\+\_\+\+Data()}}
\item Metric values and hook context are temporary -\/ valid only during callback 
\end{DoxyItemize}
