\doxysubsection{Detailed Description}
\doxylink{classlaunchdarkly_1_1server__side_1_1AllFlagsState}{All\+Flags\+State} is a snapshot of the state of multiple feature flags with regard to a specific evaluation context.

Serializing this object to JSON using boost\+::json\+::value\+\_\+from will produce the appropriate data structure for bootstrapping the Launch\+Darkly Java\+Script client.

To do this, the header \texorpdfstring{$<$}{<}\doxylink{json__all__flags__state_8hpp_source}{launchdarkly/server\+\_\+side/serialization/json\+\_\+all\+\_\+flags\+\_\+state.\+hpp}\texorpdfstring{$>$}{>} must be included to make the appropriate {\ttfamily tag\+\_\+invoke} implementations available to boost.
