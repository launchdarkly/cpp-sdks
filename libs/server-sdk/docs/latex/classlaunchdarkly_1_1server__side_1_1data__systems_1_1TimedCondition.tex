\doxysubsection{Detailed Description}
Base class for conditions that fire after a duration elapses on the orchestrator\textquotesingle{}s executor. Owns the result promise, the cancellation handle for the active timer (if any), and the state required to safely arm, cancel, and resolve the timer across threads.

Derived classes implement \mbox{\hyperlink{classlaunchdarkly_1_1server__side_1_1data__interfaces_1_1IFDv2Condition_aee8983da07be4fa3606597084ab7ff06}{Inform()}} to translate orchestrator events into arm/cancel actions on the timer. Subclasses also implement \mbox{\hyperlink{classlaunchdarkly_1_1server__side_1_1data__interfaces_1_1IFDv2Condition_a9eaddbfc83bb42bc4343e742906f134a}{Get\+Type()}} to report whether they are a fallback or recovery condition.
