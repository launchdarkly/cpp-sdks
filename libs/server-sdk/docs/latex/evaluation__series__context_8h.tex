\doxysubsection{Detailed Description}
C bindings for read-\/only evaluation context passed to hooks. 

Evaluation\+Series\+Context provides information about a flag evaluation to hook callbacks. All data is read-\/only and valid only during the callback execution.

LIFETIME\+:
\begin{DoxyItemize}
\item All context parameters are temporary -\/ valid only during callback
\item Do not store pointers to context data
\item Copy any needed data (strings, values) if you need to retain it 
\end{DoxyItemize}
