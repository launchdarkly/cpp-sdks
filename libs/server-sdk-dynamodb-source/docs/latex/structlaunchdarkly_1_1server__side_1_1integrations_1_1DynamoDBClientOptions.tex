\doxysubsection{Detailed Description}
Optional knobs for constructing the AWS Dynamo\+DB client used by \mbox{\hyperlink{classlaunchdarkly_1_1server__side_1_1integrations_1_1DynamoDBDataSource}{Dynamo\+DBData\+Source}} (and other Dynamo\+DB-\/backed integrations). 

When unset, fields fall through to the AWS SDK\textquotesingle{}s defaults\+:


\begin{DoxyItemize}
\item region resolves via the SDK region provider chain (environment, shared config file, instance metadata).
\item endpoint defaults to the standard AWS Dynamo\+DB endpoint for the resolved region. Set it to point at Dynamo\+DB Local or Local\+Stack, e.\+g. {\ttfamily \href{http://localhost:8000}{\texttt{ http\+://localhost\+:8000}}}.
\item If none of aws\+\_\+access\+\_\+key\+\_\+id / aws\+\_\+secret\+\_\+access\+\_\+key / aws\+\_\+session\+\_\+token are set, the SDK\textquotesingle{}s default credential provider chain is used (environment variables, shared credentials file, EC2/\+ECS roles). 
\end{DoxyItemize}
