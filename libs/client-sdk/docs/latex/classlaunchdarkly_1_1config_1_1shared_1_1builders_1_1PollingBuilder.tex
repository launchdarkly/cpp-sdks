Sets the filter key for the polling connection.

By default, the SDK is able to evaluate all flags in an environment.

If this is undesirable -\/ for example, because the environment contains thousands of flags, but this application only needs to evaluate a smaller, known subset -\/ then a filter may be setup in Launch\+Darkly, and the filter\textquotesingle{}s key specified here.

Evaluations for flags that aren\textquotesingle{}t part of the filtered environment will return default values.
