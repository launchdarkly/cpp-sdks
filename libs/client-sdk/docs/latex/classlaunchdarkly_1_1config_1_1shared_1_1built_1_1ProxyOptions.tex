\doxysubsection{Detailed Description}
Proxy configuration for HTTP requests.

When using CURL networking (LD\+\_\+\+CURL\+\_\+\+NETWORKING=ON), this controls proxy behavior\+:
\begin{DoxyItemize}
\item std\+::nullopt (default)\+: CURL uses environment variables (ALL\+\_\+\+PROXY, HTTP\+\_\+\+PROXY, HTTPS\+\_\+\+PROXY)
\item Non-\/empty string\+: Explicitly configured proxy URL takes precedence over environment variables
\item Empty string\+: Explicitly disables proxy, preventing environment variable usage
\end{DoxyItemize}

The empty string is forwarded to the networking implementation (CURL) which interprets it as \char`\"{}do not use any proxy, even if environment variables are set.\char`\"{}

When CURL networking is disabled, attempting to configure a proxy will throw an error.
