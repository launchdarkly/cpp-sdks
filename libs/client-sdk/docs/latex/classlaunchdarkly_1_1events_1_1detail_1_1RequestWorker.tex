\doxysubsection{Detailed Description}
\mbox{\hyperlink{classlaunchdarkly_1_1events_1_1detail_1_1RequestWorker}{Request\+Worker}} is responsible for initiating HTTP requests to deliver event payloads to Launch\+Darkly. It will attempt re-\/delivery once if an HTTP failure is deemed recoverable.

Request\+Workers are meant to be instantiated once and then used repeatedly as they become available.
