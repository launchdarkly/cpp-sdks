\doxysubsection{Detailed Description}
\mbox{\hyperlink{classlaunchdarkly_1_1events_1_1detail_1_1WorkerPool}{Worker\+Pool}} represents a pool of workers capable of delivering event payloads to Launch\+Darkly. In practice, this means initiating overlapping HTTP POSTs, assuming more than one worker is active.

A \mbox{\hyperlink{classlaunchdarkly_1_1events_1_1detail_1_1WorkerPool}{Worker\+Pool}}\textquotesingle{}s executor is meant to be executed from a single thread.
