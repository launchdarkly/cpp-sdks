\doxysubsection{Detailed Description}
Interface to allow listening for flag changes. Notification events should be distributed after the store has been updated. Meaning that the \char`\"{}new\char`\"{} value will correspond to the current value in the store, and the \char`\"{}old\char`\"{} value will be what the value was before the update.
