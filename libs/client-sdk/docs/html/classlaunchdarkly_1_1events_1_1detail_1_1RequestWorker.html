Detailed Description

RequestWorker is responsible for initiating HTTP requests to deliver event payloads to LaunchDarkly. It will attempt re-delivery once if an HTTP failure is deemed recoverable.

RequestWorkers are meant to be instantiated once and then used repeatedly as they become available.
